# 배경음악 파일

⚠️ **중요**: 음성 메시지 배경음악 기능을 사용하려면 이 폴더에 다음 5가지 음악 파일을 추가해야 합니다.

## 필요한 파일

이 폴더 (`public/bgm/`)에 다음 파일을 추가하세요:

1. `comedy.mp3` - 🤡 개그적인 음악 (코믹한 효과음)
2. `explosion.mp3` - 💥 폭발하는 음악 (드라마틱한 임팩트)
3. `epic.mp3` - 🎻 웅장한 음악 (에픽/오케스트라)
4. `lovely.mp3` - 💕 사랑스러운 음악 (달콤한 멜로디)
5. `energetic.mp3` - 🎉 신나는 음악 (업비트/파티)

## 무료 음악 다운로드 방법

### 추천 사이트:

**1. Pixabay Music** (가장 추천)
- 사이트: https://pixabay.com/music/
- 완전 무료, 저작권 걱정 없음
- 검색어 예시: "comedy", "epic", "romantic", "upbeat"
- MP3 다운로드 가능

**2. YouTube Audio Library**
- 사이트: https://www.youtube.com/audiolibrary
- 구글 계정 필요
- 분위기별 필터링 가능

**3. Incompetech**
- 사이트: https://incompetech.com/music/royalty-free/
- Kevin MacLeod의 무료 음악 라이브러리

**4. Free Music Archive**
- 사이트: https://freemusicarchive.org/

## 음악 길이 추천

각 파일은 **10-30초** 정도의 짧은 클립으로 준비하는 것을 추천합니다.
- 너무 길면 음성 메시지보다 음악이 더 길어질 수 있음
- 짧은 클립이 믹싱할 때 더 자연스러움

## 설치 후

파일을 추가한 후:
1. 앱을 새로고침
2. 음성 메시지 녹음
3. 확인 모달에서 배경음악 선택
4. 미리듣기 및 볼륨 조절
5. 전송!

🎵 음악 파일이 추가되면 모든 기능이 자동으로 작동합니다!
