
텔레그램 메신저와 동일한 기능들을 가진 어플을 만들고 싶어. 다만 서비스 이름은 Vault Messenger 로 해줘.
텔레그램과 동일하되, 추가 되는 기능은 다음과 같아 : 

명령 기능 (핵심 기능) : 채팅방 내에서 내가 명령어를 만들 수 있는 기능을 통해 채팅방내 대화 내용 중 하나를 메모처럼 저장/불러오기 할 수 있고, 업로드 파일도 저장/불러오기 할 수 있도록 하며, 하나의 채팅방에서 만든 명령어는 해당 채팅방에서만 불러오기 기능이 가능하고 다른 채팅방에서는 안되도록 함. 

* 명령 등록 (예시)
    * 사용자가 파일을 업로드하면 팝업창을 띄워 "원하는 명령어를 입력해주세요" 라는 안내와 함께 명령어를 입력받음
    * 예: 사용자가 mycode.py 파일을 올리면, 팝업에서 #runpython을 입력할 수 있음
    * 시스템은 이 명령어(runpython)와 해당 파일을 매핑하여 저장함
* 명령 호출
    * 이후 채팅창에 #runpython을 입력하면, 등록했던 mycode.py 파일을 자동으로 채팅에 다시 불러옴 (미리보기 + 다운로드 링크 제공)
* 명령어 편집/삭제
    * 등록된 명령어를 관리하는 UI 제공 (명령어 목록 확인, 수정, 삭제)
* 명령어는 중복되는경우 사용자에게 알림을 주어 다른 명령어를 선택하거나, 기존 명령어를 삭제할 수 있게 해서 중복방지.

업로드한 파일은 모두 사용자 본인이 삭제하지 않는 이상 영구저장을 하며, 메인화면에서 ‘저장소’ 라는 탭을 통해 내가 만든 모든 명령어와 파일들, 메모들을 한눈에 관리할 수 있도록 함. 



 BIZ Message 라는 이름의 메신저 프로그램을 만들려고 해. 서비스의 디자인은 전반적으로 Telegram 의 테마를 비슷하게 하되 색상만 보라색으로 선정해줘. 최대한 흰색을 많이 사용하고, 몇몇 포인트만 보라색을 사용해줘. 보라색은 그라이데이션을 준 보라색으로..

로고는 Meta AI 로고처럼 해줘, 다만 ㅅ색상은 다르게 해줘.

전체적인 UX/UI 는 Telegram 컨셉을 따르되, PC용과 모바일용을 구분지어서 만들어야 함. 
실제로 모든 기능은 작동을 해야하며, 직접 여러개의 테스트 계정을 만들어서 디거깅 혹은 테스트를 해줘.

1. 랜딩페이지 (로고 보여줌 + 로딩)
2. 로그인 페이지
    1. 전화번호 문자인증 가입 (준비중)
    2. 테스트 로그인 : 로그인시 계정 자동 생성
3. 메인페이지
    1. 탭 메뉴
        1. 연락처 : 내가 친구추가한 연락처 리스트
            1. 리스트 정렬 기능 
                1. 닉네임 순으로 정렬
                2. 이름 순으로 정렬
                3. 마지막 접속순으로 정렬
            2. 친구추가
                1. + 아이콘 클릭 후 아이디 입력 혹은 QR코드 촬영
                2. 친구 추가시 나에게 보여질 닉네임 내가 직접 입력 : 입력 안하는 경우 기본 아이디로 노출 
            3. 친구삭제
                1. PC버전 : 리스트 우측클릭 후 삭제 혹은 상위 고정 기능
                2. 모바일 버전 : 좌측으로 터치스와이프 후 삭제 혹은 상위고정 버튼 설정
                3. 공통 : 연락처 리스트 클릭시 신규 채팅방 생성. 이미 해당 연락처와 채팅방이 생성되어 있다면 그 채팅방으로 이동.
        2. 채팅방 : 이용중인 채팅방 리스트를 보여줌
            1. 채팅방 생성 기능
                1. 별도의 연락처리스트 팝업을 띄워 채널에 포함시킬 친구 등록. (다수 선택 가능)
            2. 채팅방 삭제 기능 
                1. PC버전 : 마우스 우측클릭을 통해 ‘채널삭제’
                2. 모바일 버전 : 길게 터치하여 ‘채널삭제’
            3. 채팅방 상위 고정 기능
                1. PC버전 : 마우스 우측클릭을 통해 ‘고정’
                2. 모바일 버전 : 길게 터치하여 ‘고정’
            4. 채팅방 리스트 클릭 후 실제 채팅방 입장
                1. 채팅 입력박스 좌측에 클립아이콘 (파일업로드), 태그(#) 아이콘 (클릭시 채팅박스에 # 입력) 우측에 엔터 버튼 (텍스트 입력)
                2. 암호화 실시간 메시징
                3. 메세지 자동저장
                4. 업로드 파일 바로보기 / 다운로드 기능
                5. 태그기능 : 태그기능은 메세지 혹은 업로드하는 파일에 내가 원하는 태그값을 저장하여 언제든지 다시 불러올 수 있게 하는 기능 임.
                    1. 태그는 채팅창에 # 를 입력하고 그 뒤에 설정한 태그명을 입력하면 해당 파일 및 메세지 내용 소환 기능 (소환하는 경우 일반 업로드와 디자인적 차별을 두어야 함)
                    2. 파일 업로드 할 경우 태그 기능
                    3. PC버전 : 메세지 마우스 우측클릭 후 ‘메모저장’ 누르면 태그 입력하기 나옴
                    4. 모바일 버전 : 메세지를 길게 터치하면 ‘메모저장’ 버튼이 나오고 메모저장을 하면 태그 입력하기 나옴.
                6. 파일업로드 및 영구저장 기능
                    1. PC버전 : 드랍엔드래그로 파일 업로드
                    2. 모바일버전 : 터치버튼으로 파일 업로드
                    3. 공통 : 파일 업로드시 태그(#) 설정
                7. 채팅방 부가 기능
                    1. 태그(#) 입력을 통해 업로드 파일 재소환기능
                    2. 태그(#) 입력을 통해 메세지 재생성 기능 (메세지 재생성 할때, 단순 텍스트만이 아닌, 원래 그 메세지를 남겼던 사람의 이름과 남겼던 시간을 함께 노출)
                8. 태그 (#) 기능
                    1. # 입력후 단어 입력 할때마다 자동 검색 기능 
                    2. 검색하여 선택하면, 채팅창에 해당 태그에 해당하는 파일 혹은 메세지 재소환 
                    3. 태그를 통해 자료를 재소환 하는 경우, 유저 본인만 볼 수 있도록 함.
            5. 자료실 : 모든 채팅방에 업로드 했거나 저장한 메세지를 검색할 수 있도록 함
                1. 전체 태그 검색 기능 필수
                2. 파일 공유 기능 (바로 다운받을 수 있는 링크)
                3. 태그 삭제 기능. 단, 자료실에 저장되는 모든 파일과 텍스트는 꼭 최소 하나 이상의 태그가 존재하여야 한다)
            6. 설정
                1. 내 프로필 사진 : 업로드/변경 가능
                2. QR코드 생성 기능 (모든 사용자 개인에게 부여되는 QR코드 생성)
                3. QR코드 촬영 기능 (타인의 QR코드를 촬영하여 자동으로 친구추가 기능)
                4. 내 프로필
                    1. 사용자명 변경
                    2. 전화번호 등록
                    3. 생일입력
                5. 알림 및 소리
                    1. 푸쉬 On/Off
                    2. 알림소리 선택
                6. 자료실
                    1. 저장중인 용량 그래프로 보여줌
                    2. 자료실 용량 구매 버튼
                7. 시스템 언어 선택 : 영어, 한글 선택 하면 시스템 기본 문구는 해당 언어에 맞춰 번역 후 시스템 제공
                8. 질문하기 : 준비중
