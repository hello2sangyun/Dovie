# Dovie Messenger iOS 앱

## 설치 방법

1. 압축 해제 후 App 폴더로 이동:
   ```bash
   cd App
   ```

2. CocoaPods 의존성 설치:
   ```bash
   pod install
   ```

3. Xcode에서 열기:
   ```bash
   open App.xcworkspace
   ```

## 주의사항

- App.xcodeproj가 아닌 App.xcworkspace 파일을 열어야 합니다
- pod install 후에 App.xcworkspace 파일이 생성됩니다
- iOS 14.0 이상에서 실행됩니다

## 포함된 기능

- 실시간 채팅
- 푸시 알림
- 카메라 및 파일 시스템 접근
- 햅틱 피드백
- 상태바 제어
- 토스트 알림
